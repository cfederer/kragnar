How–To: iTunes Tips

Adjust Default Volume on Songs

  1. Highlight/Right Click entire playlist
  2. Highlight/Right Click entire playlist
  3. Select “Options”
  4. Click “Volume Adjustment” box
  5. Adjust volume from the middle to ~75%

Set Up Crossfading

  1. Select “Edit”
  2. Select “Preferences”
  3. Select “Playback”
  4. Click “Crossfade Songs” box
  5. Adjust to ~5 seconds
